Las ecuaciones:

  1. Las ecuaciones son una cuenta con una o más incógnitas y se resuelven buscando el valor de las mismas.

  2. Dentro de las ecuaciones se utilizan las fórmulas que sirven para encontrar el valor de las mismas. Ejemplos con la aritmética:

    3. \[ (a+b)^2=a^2+b^2 \]

Empezando con ecuaciones:

  1. En las ecuaciones, al poner un número frente a una incógnita, significa que el número estará multiplicando a la misma. Ejemplos:
    2. \[ 4*x=4x \] \[ x=1x \] \[ -x=-1x \]
  2. En casos de tener una o más incógnitas, se resuelve moviendo las mismas hacia un solo lado del igual y lo demás del otro. Al hacer esto los signos se intercambian por el opuesto. Ejemplos:
    3. \[ x+3=5 \] \[ x=5-3 \] \[ x=2 \]
  3. Otro ejemplo más difícil:
    4. \[ 4x+2=5x-4 \] \[ 4x-5x=-4-2 \] \[ -1x=-6 \] \[ x= \frac {-6}{-1} \] \[ x=6 \]

el coeficiente de una ecuacion o incognita:

  1. el coeficiente de una incognita es el numero que lo esta multiplicando,por ejemplo:
    2. \[ \ 2x=y \]

    aqui el coeficiente seria dos porque esta multiplicando a la incognita

aprendiendo a encontrar valores:

  1. en una ecuacion los numeros se cuentan con un orden,por ejemplo:
    2. \[ \ 2x+3x-4x=20 \]

    en este caso seria asi:

    \[ \ a=2 \] \[ \ b=3 \] \[ \ c=-4 \] \[ \ d=20 \]

    entonces intercambiando los valores te queda:

    \[ \ ax+bx-cx=d \]

Aprendiendo a verificar ecuaciones:

  1. Para verificar una ecuación se intercambia la incógnita por el valor encontrado. Veamos unos ejemplos:
    2. \[ 3x+4-7x=8+2x-4 \] \[ 3x-7x-2x=8-4-4 \] \[ -6x=0 \] \[ x=\frac{0}{-6} \] \[ 3(0)+4-7(0)=8+2(0)-4 \] \[ 0+4-0=8+0-4 \] \[ 4=8-4 \] \[ 4=4 \]
  2. Al darnos lo mismo de ambos lados del igual significa que está perfectamente bien.

Potencia y raíz de ecuaciones:

  1. Al cambiar de lado un término cambia su signo al opuesto. Eso quiere decir que al implementar potencias y raíces se intercambiarán entre sí cada que los movamos por el igual. Ejemplos:
    2. \[ x^2=4 \] \[ x=\sqrt{4} \]
  2. Ejemplo contrario:
    3. \[ \sqrt{x}=4 \] \[ x=4^2 \]

Ecuaciones distributivas:

  1. Al tener una incógnita dentro de un paréntesis sumando o restando algún número natural, se tiene que distribuir lo que multiplique al paréntesis para quitarlo. Ejemplo:
    2. \[ 2(x+5)=2x+10 \] \[ 1(x+3)=1x+3 \] \[ -1(x+4)=-1x-4 \]
  2. La fórmula es:
    3. \[ a(b+c)=ab+ac \]
  3. También aplica con la división:
    4. \[ \frac{(x+3)}{2}=\frac{x}{2}+ \frac{3}{2} \]
  4. La fórmula es:
    5. \[ \frac{(a+b)}{c}= \frac{a}{c} + \frac{b}{c} \]

Elige el tipo de ejercicio:

  • Ahora veremos temas más difíciles, como tener dos diferentes incógnitas,binomios, polinomios y la factorización de los mismos.

    • dos incognitas diferentes:

    1. en este caso usaremos el metodo de sustitucion,que dice que le pongamos un valor a una incognita para que solo quede una en la ecuacion. ejemplo:
      2. \[ \ x+y=30 \]

      le ponemos un valor a x:

      \[ \ x=2y \]

      sustituimos:

      \[ \ 2y+y=30 \] \[ \ 3y=30 \] \[ \ y= \frac {30}{3} \] \[ \ y=10 \]

      y con el valor de "y" sabemos cuanto vale "x":

      \[ \ x=2y \] \[ \ x=2(10) \] \[ \ x=20 \]

      verificamos:

      \[ \ x+y=30 \] \[ \ 20+10=30 \] \[ \ 30=30 \]
    2. otro ejemplo:
      3. \[ \frac {3x-2y}{2}=2y \] \[ \ x=2y \]

      (este valor puede cambiar,solo que aqui lo usare asi)

      \[ \frac {3(2y)-2y}{2}=2y \] \[ \frac {4y}{2}=2y \] \[ \ 2y-2y=0 \] \[ \ 0=0 \]

      verificamos:

      \[ \frac {3(0)-2(0)}{2}=2(0) \] \[ \frac {0}{2}=0 \] \[ \ 0=0 \]

    binomio:

    1. un binomio se refiere a la ecuacion donde hay dos terminos elevados al cuadrado,y se resuelven de la siguiente manera:
      2. \[ \ (a+b)^2=(a+b)(a+b) \]

      ahora se multiplican los primeros dos terminos por todo el resto:

      \[ \ a^2+ab+ab+b^2=0 \]

      se suman los terminos semejantes y te queda:

      \[ \ (a+b)^2=a^2+2ab+b^2]

      (lo que queda es un polinomio que es el siguiete nivel que veremos)

    2. ahora que sabemos como resolverlo hagamos un binomio:
      3. \[ \ (x+3)^2=0 \] \[ \ (x+3)(x+3)=0 \] \[ \ x^2+3x+3x+9=0 \] \[ \ x^2+6x+9=0 \]

    polinomios:

    1. los polinomios son la suma o resta de varios terminos con un coeficiente numerico,una incognita y un exponente entero no negativo,por ejemplo:
      2. \[ \ 3x^2+2x-3x+2 \]
    2. y los polinomios se pueden resolver de dos formas principales:
    3. -formula general:
      4. \[ \ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]
    4. o factorizacion,primero veremos la formula general y luego la factorizacion.

    formula general para polinomios cuadraticos:

    1. una vez ya tenemos la formula solo hay que remplasar los valores,por ejemplo:
      2. \[ \ x^2+3x-2=0 \]

      la formula es:

      \[ \ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

      reemplazamos valores:

      \[ \ x = \frac{-3 \pm \sqrt{3^2 - 4(1)(-2)}}{2(1)} \]

      resolvemos:

      \[ \ x = \frac {-3 \pm \sqrt{9+8}}{2} \] \[ \ x = \frac {-3 \pm 4,12}{2} \] \[ \ x_1 = \frac {1,12}{2} \] \[ \ x_2 = \frac {-7,12}{2} \] \[ \ x_1 = 0,56 \] \[ \ x_2 = -3,56 \]
    2. tambien,para casos donde despues del igual no sea cero se hace esto por ejemplo:
      3. \[ \ x^2+3x-4=25 \] \[ \ x^2+3x-4-25 = 0 \]
    3. y se procede resolviendo igual que antes.

    la factorizacion de polinomios cuadraticos:

    1. para resolver esto factorizando se nesecita que "a" no sea igual a cero y que el polinomio en total sea igual a cero,ejemplo y explicacion de como resolverlo:
      2. \[ \ x^2-5x+6=0 \] \[ \ a=1 \] \[ \ b=-5 \] \[ \ c=6 \]

      ahora hay que buscar un numero que multiplicado por otro (no necesariamente igaules) de como resultado a "c",y que esos mismos numeros sumados den a "b",ejemplo:

      \[ \ -3*-2=6 \] \[ \ -3+-2=-5 \]

      entonces intercambiamos valores y nos queda:

      \[ \ x^2-5+6=(x-2)(x-3) \] \[ \ x_1=2 \qquad x_2=3 \]

    suma de polinomios:

    1. para la suma de polinomios se multiplican por su coeficiente y luego se reordenan y suman terminos similares,ejemplo:
      2. \[ \ (3x^2-2x+3-4)+(3x^3-2x^2+4x-5+14)=0 \] \[ \ 3x^2-2x+3-4+3x^3-2x^2+4x-5+14=0 \] \[ \ 3x^3+x^2+2x+5=0 \]
    2. para la resta es igual.
